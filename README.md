# Splash creator

Example:

```
    <plugin>
        <groupId>org.bitstrings.maven.plugins</groupId>
        <artifactId>splasher-maven-plugin</artifactId>
        <version>1.0.0-SNAPSHOT</version>
        <configuration>
            <outputFile>image.png</outputFile>
            <canvas>
                <width>640</width>
                <height>480</height>
                <color>#FFFFFF</color>
            </canvas>
        </configuration>
    </plugin>

```
